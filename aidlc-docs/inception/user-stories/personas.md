# ペルソナ定義

## Persona 1: 部署メンバー（田中 さん）

| 項目 | 内容 |
|---|---|
| 名前 | 田中 健太（代表的な部署メンバー） |
| 役割 | 部署メンバー（全員同一権限） |
| 技術レベル | 一般的なWebブラウザ操作ができる |
| 主な目的 | 読みたい本を素早く見つけて借り、返却期限を守る |
| 主な不満 | 誰がどの本を借りているか分からない。Excelで管理していたが更新が面倒 |
| 利用シーン | 業務中に参考書を探す、新しく買った本を登録する、借りた本を返却する |

### 代表的なジャーニー
1. ログイン → 本を検索 → 貸出申請
2. ログイン → 貸出中の本一覧を確認 → 自分の貸出本を返却
3. ログイン → 新しく届いた本を登録 → カテゴリを設定
