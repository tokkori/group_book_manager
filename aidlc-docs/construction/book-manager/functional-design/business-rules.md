# ビジネスルール定義

## 認証ルール

| ID | ルール | 条件 | 結果 | エラーコード |
|---|---|---|---|---|
| BR-AUTH-01 | ログイン失敗メッセージは汎用化 | ユーザー不存在 or パスワード不一致 | "ユーザー名またはパスワードが正しくありません" | 401 |
| BR-AUTH-02 | ユーザー名重複禁止 | 同一usernameが既存 | "このユーザー名は既に使用されています" | 400 |
| BR-AUTH-03 | パスワード最小長 | 8文字未満 | "パスワードは8文字以上で設定してください" | 422 |
| BR-AUTH-04 | JWTトークン必須 | 認証必要エンドポイントにトークンなし | "認証が必要です" | 401 |
| BR-AUTH-05 | JWTトークン有効期限 | トークン期限切れ | "セッションが期限切れです。再ログインしてください" | 401 |

## 書籍ルール

| ID | ルール | 条件 | 結果 | エラーコード |
|---|---|---|---|---|
| BR-BOOK-01 | タイトル必須 | title が空 or NULL | "タイトルは必須です" | 422 |
| BR-BOOK-02 | タイトル最大長 | title が500文字超 | "タイトルは500文字以内で入力してください" | 422 |
| BR-BOOK-03 | 出版年の範囲 | published_year < 1000 or > 現在年 | "出版年が正しくありません" | 422 |
| BR-BOOK-04 | ISBN重複警告 | 同一ISBNが既存 | 登録は続行、レスポンスに警告を付与（ブロックしない） | — |
| BR-BOOK-05 | 貸出中書籍の削除禁止 | 削除対象書籍の貸出中レコードが存在 | "貸出中の書籍は削除できません" | 400 |
| BR-BOOK-06 | 存在しない書籍への操作 | book_id が存在しない | "書籍が見つかりません" | 404 |

## 貸出ルール

| ID | ルール | 条件 | 結果 | エラーコード |
|---|---|---|---|---|
| BR-LOAN-01 | 貸出中書籍への重複貸出禁止 | 貸出対象書籍にreturn_date IS NULLのレコードが存在 | "この書籍は現在貸出中です" | 400 |
| BR-LOAN-02 | 存在しない書籍への貸出禁止 | book_id が存在しない | "書籍が見つかりません" | 404 |
| BR-LOAN-03 | 返却済み貸出への二重返却禁止 | loan.return_date が既にNOT NULL | "既に返却済みです" | 400 |
| BR-LOAN-04 | 返却は誰でも可能 | 自分が借りていなくても返却できる | 実行を許可 | — |
| BR-LOAN-05 | 貸出日は申請日の当日自動設定 | 貸出申請時 | loan_date = today() | — |
| BR-LOAN-06 | 返却日は返却処理日の当日自動設定 | 返却処理時 | return_date = today() | — |

## 貸出管理UIルール

| ID | ルール | 条件 | 結果 |
|---|---|---|---|
| BR-LOAN-UI-01 | 書籍名でソート可能 | テーブルヘッダー「書籍名」クリック | 昇順/降順で切り替え |
| BR-LOAN-UI-02 | 借り手でソート可能 | テーブルヘッダー「借り手」クリック | 昇順/降順で切り替え |
| BR-LOAN-UI-03 | 貸出日でソート可能 | テーブルヘッダー「貸出日」クリック | 昇順/降順で切り替え（初期: 降順） |
| BR-LOAN-UI-04 | 借り手で絞り込み可能 | ドロップダウンで借り手を選択 | 選択した借り手の貸出のみ表示 |

## インポートルール

| ID | ルール | 条件 | 結果 |
|---|---|---|---|
| BR-IMP-01 | 対応ファイル形式 | CSV / .xlsx / .xls 以外 | "CSV または Excel ファイルをアップロードしてください"（400） |

| BR-IMP-05 | 最大ファイルサイズ | 10MBを超えるファイル | "ファイルサイズは10MB以下にしてください"（413） |

## バリデーションルール（フロントエンド + バックエンド共通）

| フィールド | ルール |
|---|---|
| username | 必須、3〜50文字、英数字・アンダースコア・ハイフンのみ |
| password | 必須、8文字以上 |
| display_name | 必須、1〜100文字 |
| title | 必須、1〜500文字 |
| author | 任意、最大200文字 |
| isbn | 任意、最大20文字 |
| publisher | 任意、最大200文字 |
| published_year | 任意、整数、1000〜現在年 |
| location | 任意、最大200文字 |
| category name | 必須、1〜100文字 |
